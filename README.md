# QuickCrew WebPage

This is official company website for QuickCrew, which is a platform for event management and crew management. It tells everything about the company, its services, and how to get in touch with us.

## Technologies Used

- HTML
- CSS
- JavaScript
- React
- Tailwind CSS
- Vite

## Setup

To get started with the project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/quickcrew-webpage.git
   cd quickcrew-webpage
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

## Features

- **Event Management:** Tools to organize and oversee various events.
- **Crew Management:** Efficiently manage staff and volunteers.
- **Responsive Design:** Optimized for all devices using Tailwind CSS.
- **Fast Performance:** Built with Vite for a lightning-fast development experience.
